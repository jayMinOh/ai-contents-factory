# AI Contents Factory - 사용자 메뉴얼 템플릿 생성 프롬프트

## 요청 사항

아래 제품 정보를 바탕으로 **사용자 메뉴얼 템플릿**을 만들어주세요.
각 섹션마다 `[스크린샷 삽입]` 플레이스홀더를 넣어주세요. 실제 화면 캡쳐는 제가 직접 추가할 예정입니다.

---

## 제품 정보

### 제품명
**AI Contents Factory** (AI 콘텐츠 팩토리)

### 제품 설명
SNS 마케팅 영상을 AI가 분석하고, 분석 결과를 바탕으로 마케팅 이미지를 자동 생성하는 올인원 콘텐츠 제작 플랫폼

### 타겟 사용자
- 화장품/뷰티 브랜드 마케터
- SNS 콘텐츠 크리에이터
- 중소 브랜드 마케팅 담당자
- 광고 에이전시

---

## 메뉴얼에 포함할 기능별 섹션

### 1. 시작하기
- 회원가입 및 로그인 방법
- 관리자 승인 대기 안내
- 대시보드 첫 화면 소개

### 2. 브랜드/제품 등록
- 브랜드 프로필 생성 방법
- 제품 정보 입력 (제품명, 성분, 효능, 타겟 피부타입)
- 제품 이미지 업로드

### 3. 레퍼런스 영상 분석
- 지원 플랫폼: YouTube, Instagram Reels
- SNS URL 입력 방법
- 분석 결과 확인하기
  - 타임라인 세그먼트 분석
  - 훅 포인트 (시청자 관심 순간)
  - 엣지 포인트 (차별화 요소)
  - 감정적 트리거
  - 페인 포인트 (고객 공감 포인트)
- 분석 점수 해석 가이드

### 4. AI 콘텐츠 생성
#### 4-1. 캐러셀 이미지 생성
- 레퍼런스 선택
- AI 컨셉 자동 생성
- 스토리보드 생성 및 편집
- 배치 이미지 생성
- 결과물 다운로드

#### 4-2. 싱글 이미지 생성
- 레퍼런스 선택
- AI 컨셉 생성
- 이미지 생성 및 편집
- 결과물 다운로드

### 5. 이미지 편집
- 캔버스 편집 도구 사용법
- 제품 이미지 + AI 배경 합성
- 스타일 레퍼런스 적용 방법
- 텍스트/요소 추가

### 6. 프로젝트 관리
- 생성된 콘텐츠 목록 확인
- 프로젝트 저장 및 불러오기
- 콘텐츠 내보내기

## 메뉴얼 작성 가이드라인

### 톤 앤 매너
- 친절하고 쉬운 설명
- 마케터/크리에이터 관점에서 작성
- 전문 용어는 간단히 설명 추가

### 형식
- 각 섹션 시작에 해당 기능의 목적 한줄 설명
- 단계별 번호 매기기 (Step 1, Step 2...)
- 각 단계마다 `[스크린샷: 설명]` 플레이스홀더
- 팁/주의사항은 별도 박스로 구분

### 예시 형식
```
## 3. 레퍼런스 영상 분석

경쟁사나 벤치마킹할 SNS 영상을 AI가 자동으로 분석해 인사이트를 추출합니다.

### Step 1: 분석할 영상 URL 입력
[스크린샷: URL 입력 화면]
- YouTube 또는 Instagram Reels URL을 입력합니다
- 붙여넣기 후 "분석 시작" 버튼을 클릭합니다

### Step 2: 분석 결과 확인
[스크린샷: 분석 결과 대시보드]
- 타임라인별 장면 분석 결과를 확인합니다
- 각 항목의 점수와 상세 설명을 검토합니다

> 💡 팁: 훅 포인트 점수가 높은 구간을 참고하면 효과적인 콘텐츠를 만들 수 있습니다.
```
