# AI Contents Factory - 포트폴리오 PPT 생성 프롬프트

## 프로젝트 개요

아래 내용을 바탕으로 **투자자/클라이언트 대상 프레젠테이션 PPT**를 만들어주세요.
10-15장 분량으로, 시각적으로 깔끔하고 전문적인 톤으로 작성해주세요.

---

## 프로젝트 정보

### 프로젝트명
**AI Contents Factory** (AI 콘텐츠 팩토리)

### 한줄 소개
SNS 마케팅 영상을 AI가 분석하고, 분석 결과를 바탕으로 마케팅 이미지/영상을 자동 생성하는 올인원 콘텐츠 제작 플랫폼

### 타겟 사용자
- 화장품/뷰티 브랜드 마케터
- SNS 콘텐츠 크리에이터
- 중소 브랜드 마케팅 담당자
- 광고 에이전시

---

## 핵심 기능 (실제 구현 완료)

### 1. SNS 영상 분석 시스템
- **지원 플랫폼**: YouTube, Instagram Reels, TikTok
- **분석 항목**:
  - 타임라인 세그먼트 분석 (장면별 시각적 설명, 자막, 참여도 점수)
  - 훅 포인트 감지 (시청자 관심을 끄는 핵심 순간)
  - 엣지 포인트 분석 (차별화 요소)
  - 감정적 트리거 분석 (감정 유발 순간)
  - 페인 포인트 분석 (고객 공감 포인트)
- **AI 모델**: Google Gemini 3 Flash
- **점수화 시스템**: 각 분석 항목별 상세 점수 breakdown 제공

### 2. AI 기반 마케팅 이미지 생성
- **콘텐츠 타입**:
  - 캐러셀 이미지 (인스타그램 슬라이드용)
  - 싱글 이미지 (피드 포스트용)
  - 스토리/릴스 이미지 (세로형 9:16)
- **기능**:
  - 레퍼런스 분석 결과 기반 컨셉 자동 생성
  - 제품 이미지 + AI 생성 배경 합성
  - 스타일 레퍼런스 적용 (참조 이미지 스타일 전이)
  - 실시간 이미지 편집 (Fabric.js 캔버스)
- **AI 모델**: Google Gemini (이미지 생성)

### 3. AI 비디오 스토리보드 생성
- **기능**:
  - 분석된 레퍼런스 기반 스토리보드 자동 생성
  - 장면별 프롬프트 생성
  - 씬 이미지 AI 생성
  - 비디오 생성 (Veo API 연동)
- **워크플로우**: 레퍼런스 분석 → 컨셉 생성 → 스토리보드 → 이미지 생성 → 비디오 생성

### 4. 브랜드/제품 관리
- 브랜드 프로필 등록
- 제품 정보 관리 (성분, 효능, 타겟 피부타입)
- 제품 이미지 업로드 및 관리

### 5. 사용자 인증 및 관리
- JWT 기반 인증 시스템
- 관리자 승인 워크플로우
- 사용자별 프로젝트 관리

## 기술 스택

### Frontend
- **Framework**: Next.js 14 (App Router)
- **UI**: Tailwind CSS, Lucide Icons
- **상태관리**: Zustand, React Query
- **캔버스**: Fabric.js (이미지 편집)
- **언어**: TypeScript

### Backend
- **Framework**: FastAPI (Python 3.11)
- **Database**: MariaDB/MySQL (SQLAlchemy ORM)
- **캐시**: Redis
- **벡터DB**: Qdrant (유사 콘텐츠 검색)

### AI/ML
- **영상 분석**: Google Gemini 3 Flash
- **이미지 생성**: Google Gemini (Imagen)
- **비디오 생성**: Google Veo API
- **자연어처리**: LangChain

### 인프라
- **Backend**: Railway (Docker)
- **Frontend**: Vercel
- **Storage**: S3 호환 스토리지

## 주요 화면 구성

### 1. 대시보드 (/)
- 주간 생성 콘텐츠 수
- 총 콘텐츠 수
- 저장된 레퍼런스 수
- 최근 작업 목록

### 2. 레퍼런스 분석 (/references)
- SNS URL 입력
- 영상 자동 다운로드
- AI 분석 결과 대시보드
- 분석 항목별 상세 점수

### 3. 콘텐츠 생성 (/create)
- 레퍼런스 선택
- AI 컨셉 생성
- 이미지 타입 선택 (캐러셀/싱글/스토리)
- 실시간 이미지 편집
- 배치 이미지 생성

### 4. 스튜디오 (/studio)
- 스토리보드 생성
- 씬별 이미지 생성
- 비디오 생성 및 편집

### 5. 히스토리 (/history)
- 생성된 프로젝트 목록
- 이미지 다운로드
- 프로젝트 상세 보기

### 6. 관리자 (/admin)
- 사용자 승인/관리
- 시스템 설정

## 차별화 포인트

### 1. 분석 → 생성 원스톱 워크플로우
경쟁사 대비 레퍼런스 분석부터 콘텐츠 생성까지 하나의 플랫폼에서 완결

### 2. 화장품 특화 AI 분석
뷰티/화장품 마케팅에 최적화된 분석 항목 및 점수화 시스템

### 3. 상세한 점수 breakdown
단순 점수가 아닌 항목별 가중치, 이유, 개선점 제시

### 4. 스타일 레퍼런스 전이
참조 이미지의 스타일을 새로운 콘텐츠에 적용

### 5. 실시간 편집 기능
생성된 이미지를 캔버스에서 바로 편집 가능
